# Unconstrained-Plasticity-Disrupts-Memory-Consolidation-in-a-Mouse-Model-of-Rett-Syndrome
Source codes for the manuscript entitled “Unconstrained Plasticity Disrupts Memory Consolidation in a Mouse Model of Rett Syndrome” submitted to Nature Communications. The preprint is available at: https://www.biorxiv.org/content/10.64898/2026.01.29.702595v1.
## Requirements
All codes used to generate the figures were written and implemented in MATLAB (R2024a).
## Instructions for use
The experimental and computational modeling datasets are available from the corresponding authors upon request. Shared functions used in multiple analyses are provided in this repository. For figure panels that require multiple MATLAB scripts, step-by-step instructions are provided in separate instructions.
